# Roblox Pulse

A polished Roblox fan site with the latest update summary, community opinion, a Robux tax calculator, and a daily top game list.

## Files

- `index.html` — main site layout and content
- `styles.css` — visual design, typography, and responsive layout
- `script.js` — dynamic rendering for updates, community opinion, game ranking, and calculator logic

## Instructions

Open `index.html` in your browser to view the site.

## Notes

- The site now integrates live Roblox Dev Forum updates directly from `https://devforum.roblox.com/latest.json`.
- If the live feed cannot load due to network or CORS restrictions, the site falls back to sample update cards.
- The daily top games section uses the browser date to show when the list was last refreshed.
